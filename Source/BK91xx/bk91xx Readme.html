bk91xx Instrument Driver Readme

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play
Manufacturer: B+K Precision
Supported Language(s): LabVIEW 8.0
Supported Model(s): 9120, 9121, 9122, 9123
Model(s) Tested: 9120, 9123
Interface(s): GPIB, Serial
Firmware Revision(s) Tested: 1.0_1.0

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 1.0
Original Release Date: 08/21/2006
Current Revision Date: 08/21/2006

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 3.0 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to instrument.drivers@ni.com.

Note: Following functions included in standard DC Power Supply Template are not supported by this version of BK91xx instrument drivers.

• Self Test.vi

• Reset Output Protection.vi

For further details regarding this issues, please read de BK91xx Instruction Manual.
